# COMmunity (FE)

## 🚀 프로젝트 소개
- **프로젝트 설명**: KTB 실습 과제인 간단한 커뮤니티 사이트 입니다.
- **기술 스택**: JS, React, Vite
- [server repo](https://github.com/100-hours-a-week/2-swan-song-community-be)

<br>

## 📂 프로젝트 구조
```plaintext
src/
├── assets/          # 이미지, 폰트 등 정적 파일
├── components/      # 재사용 가능한 컴포넌트
├── constants/       # 설정용 상수 
├── contexts/        # 전역 상태 
├── hooks/           # 커스텀 훅
├── pages/           # 페이지별 컴포넌트
└── utils/           # 공통 유틸리티 함수
```
